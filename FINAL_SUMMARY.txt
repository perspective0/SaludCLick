📊 ENDPOINTS API
═════════════════════════════════════════════════════════════════════════════

  AUTENTICACIÓN
  POST /api/auth/register    - Crear usuario
  POST /api/auth/login       - Login
  POST /api/auth/logout      - Logout
  GET  /api/auth/me          - Usuario actual

  MÉDICOS
  GET  /api/doctors          - Listado (con filtros)
  GET  /api/doctors/:id      - Detalles
  GET  /api/doctors/:id/calendar
  GET  /api/doctors/:id/patients

  CITAS
  POST /api/appointments     - Crear cita
  GET  /api/appointments     - Listar citas
  GET  /api/appointments/:id - Detalles
  PUT  /api/appointments/:id - Actualizar
  DELETE /api/appointments/:id - Cancelar

  REGISTROS MÉDICOS
  POST /api/medical-records  - Crear registro
  GET  /api/medical-records/:patientId - Historial
  POST /api/prescriptions    - Crear receta
  GET  /api/prescriptions/patient/:patientId

  ADMIN
  GET  /api/admin/stats      - Estadísticas
  GET  /api/admin/users      - Listar usuarios
